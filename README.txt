Навигатор национальных проектов — версия v11

Файлы сайта:
- index.html — главная
- about.html — как устроены национальные проекты
- projects.html — каталог проектов
- measures.html — меры и возможности
- spheres.html — сферы жизни
- analytics.html — данные и общественное мнение
- implementation.html — как проверить реализацию
- results.html — практические результаты
- research.html — исследования и аналитика
- sources.html — источники

Изображения лежат локально в папке images. В этой версии декоративные SVG заменены на смысловые PNG-иллюстрации под конкретные страницы и блоки.

Как открыть: дважды нажать index.html.
Как опубликовать: загрузить все файлы в GitHub Pages репозиторий с заменой старой версии.
