Навигатор национальных проектов

Как открыть сайт:
1. Распакуйте архив.
2. Откройте файл index.html в браузере.

Что внутри:
- index.html — основная страница сайта;
- style.css — оформление;
- script.js — интерактив, карточки, диаграммы и мини-опрос;
- images — локальные изображения для сайта.

Как выложить на GitHub Pages:
1. Создайте новый репозиторий на GitHub.
2. Загрузите в репозиторий файлы index.html, style.css, script.js и папку images.
3. Откройте Settings → Pages.
4. В разделе Branch выберите main / root и нажмите Save.
5. Через 1–3 минуты GitHub выдаст ссылку на сайт.

Важно:
- Все изображения подключены локально из папки images.
- Сайт можно открыть как файлом на компьютере, так и опубликовать по ссылке через GitHub Pages.
- Источники данных указаны в нижнем разделе сайта.
